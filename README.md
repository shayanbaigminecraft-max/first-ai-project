# First AI Project

An AI chatbot project built with Python.

## enviroment creation 

``` conda create -n first-ai python=3.11 -y ```

## Features

- AI Chatbot
- FastAPI Backend
- LangChain Integration
- Mistralai API
- Modular Project Structure
